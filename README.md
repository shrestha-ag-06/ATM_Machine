## 📌 Project Overview

The **ATM Machine System** is a Java-based console application that simulates the working of a real-world Automated Teller Machine.
It allows users to securely access their bank account and perform essential banking operations like deposits, withdrawals, and balance inquiriesThis project demonstrates core Java concepts such as **Object-Oriented Programming (OOP), file handling, and user interaction**.

---

## ✨ Features

✅ Secure Login System (Account Number & PIN)  
✅ Deposit Money  
✅ Withdraw Money  
✅ Check Account Balance  
✅ Simple & User-Friendly Interface  
✅ Error Handling for Invalid Inputs

## ⚙️ Installation & Setup

### 🔹 Prerequisites
- Java JDK 8 or higher installed
- Any IDE (IntelliJ / Eclipse / VS Code)

---

### 🔹 Steps to Run
```text
1. Clone the Repository
git clone https://github.com/your-username/ATM-Machine.git
2. Navigate to Project Folder
cd ATM-Machine
3.Compile the Program
javac ATM.java
4.Run the Program
java ATM
```
